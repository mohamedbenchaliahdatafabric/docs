## Provide your Datadog credentials

You will need to add the following environment variables in order to use
Datadog in kozmo:
* DD_API_KEY: your datadog api key
* DD_APP_KEY: your datadog app key

## Using Datadog in Kozmo

You can now call Datadog helper functions in your Kozmo blocks to create or update metrics.
You can see the existing helper functions [here](https://github.com/kozmoai/kozmoai/blob/master/kozmo_ai/services/datadog/__init__.py).

Example:
```python
from kozmo_ai.services import datadog as dd

dd.increment('<your metric here>', tags=dict(pipeline='<pipeline_uuid>'))
```
