## Written Tutorials

### [Simplifying Data Pipelines With Kozmo Data Integrations](https://ethanbrown.substack.com/p/simplifying-data-pipelines-with-kozmo)
Ethan Brown introduces Kozmo Data Integration Blocks, bridging no-code simplicity with standard pipeline capabilities for seamless data integration across various sources. From Postgres to Salesforce, Ethan showcases the effortless process of pulling and leveraging data within standard pipelines, promising streamlined development and efficient syncs. Explore the future of data integration with Kozmo, where complexity meets accessibility.

### [Kozmo.ai â€“ Tutorials and Intro](https://janzednicek.cz/en/category/kozmoai-etl-tool/)
Jan provides a series of tutorials from intro and installation to building pipelines and charts.

### [How to Streamline Communication in Data Pipelines Using Kozmo](https://medium.com/@xiaoxugao/1ab2a885dd73)

In her post, Xiaoxu introduces the concept of a _communication lifecycle_ for data pipelines. Proactive feedback to stakeholders is essential for establishing trust and building confidence in data. Xiaoxu demonstrates how Kozmo's [callbacks](/design/blocks/callbacks) can be used to automate tasks based on block status, reducing the communication lifecycle and establishing a healthy relationship between data owners and consumers.

### [Modern Data Engineering with Kozmo](https://www.analyticsvidhya.com/blog/2023/06/modern-data-engineering-with-kozmo-empowering-efficient-data-processing/)

Sri Nikitha walks through a demo of Kozmo: from installation to a sample pipeline that loads data from a NYC taxi dataset, transforms it, and exports to BigQuery.

### [Postgres to Snowflake Data Transfer](https://sudsy-pail-0c7.notion.site/Data-Migration-Tutorial-using-Kozmo-46c027b4095d4c9c8216c566eee3c3a4)

Jafar Sharif walks through a tutorial on how to transfer data from Postgres to Snowflake using Kozmo, including some core ETL functionality. A quick and fun read!
