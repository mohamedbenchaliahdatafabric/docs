## Style guide

Kozmo follows the [Airbnb JavaScript style guide](https://airbnb.io/javascript/react/).

### Linter

To run the linter locally, execute this script:

```bash
./scripts/test.sh
```

## Setup

```bash
yarn global add eslint-config-next
yarn global add eslint_d
```
